【杂谈】不应被遗忘的“差评”佳作-亡国的阿基德

2020-06-12

Code Geass系列是自06年开始连载的原创动画系列，并有外传小说和漫画等诸多衍生作品。

截止2020年，除了众所周知的两季TV动画“叛逆的鲁路修”与“叛逆的鲁路修R2”，本系列还有前传性质的“漆黑的莲夜”，时间线穿插两季TV动画之间的“亡国的阿基德”与“双貌的奥兹”，以及作为其延伸作品的“娜娜莉的梦魇”与“反攻的朱雀”。

其中两季TV动画于17年-18年重制成了三部剧场版，并在19年初推出了其后续的“复活的鲁路修”。

鲁路修·兰佩路基，为该系列最主要的主人公之一，亦是ACG领域内公认的超人气角色。

“亡国的阿基德”，12年-16年推出的“叛逆的鲁路修”外传性质的剧场版动画，共五章。

本作的主角是EU军所属，来自11区的少年日向瑛斗。而原作的主人公鲁路修在本作只有客串的戏份。

本作时间线设定在TV版两季之间，主战场转移到了EU欧洲战线。可以说，“亡国的阿基德”虽然与“叛逆的鲁路修”属于同一个系列，且时间线保持一致，但两者在剧情上并无太多关联，可以当成单独的作品来观看。

很大一部分“叛逆的鲁路修”原作党追番的动力便是其极具个人魅力的主人公鲁路修·兰佩路基。而本作宣发海报中虽添加了疑似鲁路修的形象，但实际动画中鲁路修的登场时间颇短，且与原作主线毫不相干，因此受到一些原作党的差评。
